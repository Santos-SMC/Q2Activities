Linked Styles

External style sheet style sheets or linked styles are made outside he HTML file. An external style sheet is linked via <link>. The <link> element allows you to establish document relationships.

The <link> tells the browser to find the specified style sheet. It can only be used within the <head></head> section of the document. The style sheet file does not cotain any HTML code; it contains only style rules.
